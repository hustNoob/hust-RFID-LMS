图书馆RFID智能管理系统

一、开发环境
1. 硬件环境：
   - Windows 11PC电脑
   - 一台兼容ISO 15693协议的UHF RFID读卡器（配套驱动及USB/串口线）
   - RFID标签卡15693

2. 开发工具版本：
   - Qt Creator 5.0 及以上
   - Qt 5.5
   - 编程语言：C++

3. 数据库管理：
   - 本系统使用 SQLite 3（推荐3.7及以上，Qt自带该库）
   - 数据库存储文件为 database.db，由程序自动创建和维护，无需手动配置


二、系统实现功能简介
1. 串口自动识别、连接与断开，支持多种波特率配置，支持RFID读卡器。
2. RFID刷卡自动识别：
   - 用户卡刷卡后自动显示个人信息和借书历史
   - 图书卡刷卡后自动关联借还书流程，校验借阅状态、超期等
3. 借书、还书全流程管理，支持校验、数据库自动更新、表格实时刷新、超期提示
4. 用户管理（注册、修改、删除、挂失、取消挂失），支持刷卡快速录入和查找
5. 图书管理（添加书籍、信息维护、删除、批量查询），支持刷卡录入，状态管理
6. 借书记录管理（按用户、图书编号检索、删除历史记录等）
7. 数据所有操作持久化数据库，多表关联、数据防丢失
8. 界面所有交互友好，自动填表、结果反馈、错误提示完善，完全支持鼠标键盘和刷卡输入



三、安装与部署步骤
1. 确保qt环境无问题
2. 将LibraryManagementSystem文件夹放到一个无任何中文目录的路径下，不然无法运行
3. 点击LMS.pro进入qt项目
4. 点击左下角“运行”，快捷键为Ctrl＋R，启动项目
5. 点击连接，即可进入系统操作
6. 项目生成的数据库文件位置：E:\exps\rfid\U202115432-滕德彬\LibraryManagementSystem\build-LMS-Desktop_Qt_5_5_0_MinGW_32bit-Debug\database.db
